## 🎯 The Problem

Banks spend **thousands of man-hours** every year:
- Manually reading regulatory circulars (Basel, Fed, EBA, DORA, FinCEN)
- Converting them into action items
- Emailing departments and chasing for updates
- Manually verifying completion

**ARIC automates all of this with 4 AI agents.**

---

## 🤖 How It Works

```
Regulation Published
        ↓
🔍 Monitor Agent    →  Detects new regulations from 14+ sources
        ↓
🧠 Parser Agent     →  Reads regulation → Generates MAPs using Llama 3.3
        ↓
📋 Assigner Agent   →  Routes each MAP to the correct bank department
        ↓
✅ Validator Agent  →  Autonomously verifies completion evidence
        ↓
📊 Compliance Score Updated — Zero human intervention
```

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔍 **Monitor Agent** | Watches 14+ regulatory feeds (Basel IV, Fed Reserve, EBA, DORA, FinCEN) |
| 🧠 **Parser Agent** | Reads regulation PDFs → Generates Measurable Action Points using Groq/Llama-3.3-70b |
| 📋 **Assigner Agent** | Intelligently routes MAPs to Risk, Compliance, IT, Legal, Treasury, Operations |
| ✅ **Validator Agent** | Autonomous completion validation via document hash, system API & portal cross-check |
| 🤖 **Full Pipeline API** | One API call triggers all 4 agents in sequence |
| 📊 **AI Compliance Report** | Board-level executive summary generated by AI |
| 📡 **Live Dashboard** | Real-time metrics, MAP status, compliance score trend |
| 🏢 **Department Workloads** | Visual workload distribution across all bank departments |
| 📑 **Audit Log** | Complete agent activity trail for regulatory audit purposes |

## 🚀 Quick Start

### Prerequisites
- Node.js v18+
- Python 3.10+
- Groq API Key (free at [console.groq.com](https://console.groq.com))

### 1. Clone the repo
```bash
git clone https://github.com/Karandaiya88/aric-compliance-platform.git
cd aric-compliance-platform
```

### 2. Frontend Setup
```bash
npm install
npm run dev
# Opens at http://localhost:3000
```

### 3. Backend Setup
```bash
cd backend
python -m venv venv

# Windows
venv\Scripts\activate

# Mac/Linux
source venv/bin/activate

pip install -r requirements.txt
```

### 4. Environment Variables
```bash
# Create backend/.env
DATABASE_URL=sqlite:///./aric.db
GROQ_API_KEY=your_groq_api_key_here
APP_ENV=development
```

### 5. Seed Database & Run
```bash
python seed.py        # Load sample data
uvicorn main:app --reload
# API at http://localhost:8000
# Docs at http://localhost:8000/docs
```

## 📡 API Endpoints

### Regulations
```
GET    /api/regulations/          → Get all regulations
POST   /api/regulations/          → Add new regulation
PATCH  /api/regulations/{id}      → Update status
GET    /api/regulations/stats/summary → Summary stats
```

### MAPs (Measurable Action Points)
```
GET    /api/maps/                 → Get all MAPs
POST   /api/maps/                 → Create MAP
PATCH  /api/maps/{id}             → Update progress
POST   /api/maps/{id}/validate    → Mark as validated
GET    /api/maps/stats/summary    → Status breakdown
```

### AI Agents
```
POST   /api/agents/monitor        → Trigger Monitor Agent
POST   /api/agents/parse/{id}     → Trigger Parser + Assigner
POST   /api/agents/validate       → Run Validation Sweep
POST   /api/agents/pipeline/run   → Full 4-agent pipeline
GET    /api/agents/report         → AI Compliance Report
```
